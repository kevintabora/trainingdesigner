Getting Started

Training Designer

Training Designer helps you build structured, well-organized training programs. You break each course down into focused learning activities, each tied to a specific objective, so your program has clear purpose from day one.

What you can do
📅
Build day-by-day
Organize activities into training days. The app tracks totals and schedules automatically.
📈
Analyze your program
Charts and tables show how time is split across cognitive tasks, activities, delivery methods, and more.
🎯
Structured objectives
Every activity is linked to a measurable learning objective and a cognitive task level.
📄
Excel-first workflow
Prefer planning in a spreadsheet? Use the blank template, then load it right into the app.
How it works
  • Click + Add Course in the top menu to open the form. Fill in your course details and learning activities.
  • Click Submit. Each learning activity becomes its own row in the day table.
  • Edit, reorder, copy, or delete rows as your plan takes shape.
  • Click Report in the menu bar to jump to the Report section and see how your program breaks down.
  • When you're ready, go to File → Save to export everything as an Excel file.
  • Need help? Go to File → Guide for a full walkthrough of every feature.
Building Courses

Creating a Course

Each time you submit the form, the app creates one or more rows in your day table. Every learning activity you define becomes its own row.

Opening the form

Click + Add Course in the top menu bar. A form will appear where you can fill in the course details (day, unit, module) and at least one learning activity. When everything looks right, click Submit.

Adding multiple activities

A single course can contain several learning activities. For example, a module on "Creating a Case in Salesforce" might include a lecture, a navigation practice, and a knowledge check, each as separate activities.

Click + Learning Activity inside the form to add more. Each one gets its own objective, cognitive task, and duration, and each one becomes a separate row in the table.

Tip: Use the drag handle (☰) next to each activity block to reorder them inside the form before you submit.
Editing an existing row

Click the ✏️ Edit button on any row. The form reopens with that row's data already filled in. Make your changes and submit to update it.

While you're editing, all other table actions (drag, delete, copy) are disabled. Submit or cancel the edit to re-enable them.
Building Courses

Field Reference

A breakdown of every field in the Add Course form, what it means, and what to enter.

Course-level fields

These apply to the whole course and are shared by all activities under it.

DayWhich training day this course falls on. Enter a number (e.g., 1, 2, 3).
Unit (Main Topic)The broad topic that groups related content together. Example: "Salesforce", "Print Products", "Tax Form 1065".
Module (Subtopic)A specific subject within the unit. Example: "Creating a Case in Salesforce", "Running Meetings".
Activity-level fields

Each learning activity has its own set of these fields.

DurationHow long this activity takes, in minutes.
Link (optional)A URL to an external resource. If provided, it shows as a clickable link in the Media column. Must start with http:// or https://.
PlanThe intended action for this row when reviewing or updating the program. Rows marked Remove are excluded from total duration calculations.
Notes (optional)Any extra context that may affect delivery or future updates. Only visible in the exported Excel file.
Learning Objective

A clear, measurable statement of what learners should be able to do after completing this activity. Write it as an action. Examples:

  • Verify customer identity using the two-step authentication process
  • Locate specific legal cases using Westlaw's search tools
  • Explain what common error messages mean during financial statement uploads
Cognitive Task

The level of thinking this activity requires. These build on each other, from simple recall to complex creation.

RememberRecall facts, rules, or steps exactly as learned. Example: List the required fields on a case form.
ExplainDescribe how something works or restate it in your own words. Example: Explain why a case gets escalated.
ApplyUse what you've learned to complete a task in a new situation. Example: Create a new case following the standard workflow.
EvaluateAnalyze a situation and make a judgment call. Example: Decide whether to escalate or resolve a case based on its details.
CreateCombine concepts to develop something original. Example: Design a custom workflow for handling a new case type.
Learner Activity

What the learner will actually be doing during this activity. Activities are organized into six groups based on the learner's cognitive role — not by topic or task level.

Receptive Instruction

The learner receives and absorbs information without being required to produce, decide, or perform.

Lecture Participation Video / Recording Review Reading Podcast / Audio Review eLearning - Read, Watch, Listen
Meaning Making

The learner actively organizes, connects, and makes sense of received information through reflection, questioning, or restructuring.

eLearning - Interactive Q&A with Instructor / Expert Recap & Debrief Reflection Facilitated Group Activity Concept Mapping / Mind Mapping Pre-Training / Advance Organizer
Guided Practice

The learner performs a task or applies a skill with structure and support available before being expected to work independently.

Worked Example Review eLearning - Simulate & Practice Software / Tool Practice Documentation Practice Job Aid & Reference Lookup Practice Shadowing & Job Observation
Applied & Social Learning

The learner performs in realistic, collaborative, or reduced-scaffolding contexts where peers, scenarios, or real work are the medium of instruction.

Role Play / Mock Conversation Scenario-Based Exercise Live Group Discussion Asynchronous Group Discussion Peer Teaching Live Customer Practice Mentorship / Coaching Meet & Greet Icebreaker
Assessment

The learner demonstrates knowledge or skill for the purpose of verification rather than development, ranging from low-stakes checks to high-stakes summative evaluation.

Knowledge Check Self Assessment Peer Assessment Procedural Skills Assessment Written Graded Assessment Graded Practice Conversation
Engagement & Motivation

These activities serve social, emotional, and motivational functions that enable learning rather than constitute it directly.

Learning Game Meet & Greet Icebreaker
Delivery Method

How the learning experience is structured and delivered to the learner.

Instructor-LedA trainer guides the session in real time with direct interaction and feedback.
Self-PacedLearners work through the material independently, at their own speed.
On-the-Job TrainingLearning happens through supervised practice in actual work situations.
MicrolearningShort, focused lessons designed to be completed quickly and applied right away.
Media

The material or platform the learner interacts with during this activity. Media options are organized into six groups.

Documents & Text

Static text-based materials the learner reads, references, or writes in.

Text Document Spreadsheet Activity Sheet / Workbook Online Article / Web Page
Visual Presentation

Slide-based or graphic materials that communicate primarily through visual organization.

Slideshow Infographic / Visual Aid
Audio & Video

Pre-produced one-way media the learner watches or listens to.

Audio Recording / Podcast Recorded Lecture Produced Instructional Video
Interactive Digital

Digital tools and platforms that require active learner input and respond with feedback or branching.

eLearning Module Quiz / Survey Live System / Application Digital Workspace / Notebook
Collaborative & Social

Shared platforms and channels that enable learners to co-create, discuss, and learn with others.

Collaborative Platform Discussion Channel
Live & Experiential

Real-time human or immersive media that cannot be replicated through recorded or digital formats.

Guest Speaker / SME Session VR / AR
Content Type

What kind of knowledge or skill this activity covers. The type affects how it maps against cognitive tasks in the report.

Facts / Concepts Info Foundational knowledge that needs to be remembered and understood.
Workflows Info Multi-step processes that show how tasks or roles connect end-to-end.
Software / Tools Procedural Step-by-step instructions for using a specific application or tool.
Communication Skills Procedural Standard protocols for handling customer interactions consistently.
Software / Tools Problem-Solving Troubleshooting principles for diagnosing and fixing technical issues.
Communication Skills Problem-Solving Strategies for handling unique or complex customer situations.
Managing Your Plan

Activities & Rows

Every submitted activity becomes a row in the day table. You can edit, reorder, duplicate, or remove any row at any time.

Row action buttons

Each row has four action buttons on the right side.

✏️ EditReopens the form with this row's data so you can modify any field.
📋 CopyCreates an exact duplicate of the row directly below it.
🗑️ DeletePermanently removes the row from your training plan.
☰ DragClick and hold to move a row up or down within its day, or drag it into a different day's table.
Selecting rows & bulk actions

Use the checkbox on any row to select it. You can also check the box in the day header to select every row in that day at once.

Once rows are selected, a Manage Selected dropdown appears in the menu bar with these options:

Copy Selected Move Selected Delete Selected
While rows are selected, individual row actions (edit, copy, delete, drag) and the Add Course button are disabled. Deselect all rows to re-enable them.
Table visual indicators
Dashed lineActivities that share the same unit and module are grouped together with dashed borders between them.
Solid lineA solid border marks where one course group ends and another begins.
Gray rowA scheduled break. Break rows can be edited (to change duration), copied, deleted, and dragged just like regular rows.
Breaks

Click + Add Break below any day table to insert a break row. You'll be asked for the duration in minutes. Breaks can be repositioned by dragging.

Use Hide Breaks in the menu to temporarily collapse all break rows from view. Use Delete Breaks to remove them all at once.

Total Duration

Shown at the bottom of each day table. This is the sum of all activity durations for that day, displayed as both minutes and decimal hours (e.g., 240 min | 4 hrs). Rows with the Plan set to Remove are automatically excluded from this total.

Undo

The ↺ button in the top menu bar lets you undo the last action. You can also press Ctrl+Z (or Cmd+Z on Mac). The app keeps up to 20 levels of undo history.

Undo covers: adding, editing, or deleting activities, copying or moving rows, reordering via drag-and-drop, adding or deleting breaks, changing start time, editing product/program names, resetting, and loading a file.

Undo is disabled while you're adding or editing a course. Submit or cancel the form first.
Selection tally

When you select rows using checkboxes, a floating counter appears on the right side of the screen showing the total number of selected rows and their combined duration in minutes and hours. Rows marked as Remove in the Plan column are excluded from the duration total.

Managing Your Plan

Saving & Loading

Export your training plan as a structured Excel file, or load a previously saved file to pick up where you left off.

How to save

Go to File → Save in the top menu. The save dialog lets you optionally include a date, time, and time zone in the exported file.

Before saving, make sure to set your Product Name and Training Program name. You can do this by clicking the name text next to the Design or Report header. A modal will appear where you can enter or update both names. You can save without names, but the Excel fields will be empty.

Add Date & TimeCheck this box to include a start date, time, and time zone in your exported file. Useful if your team needs a concrete schedule.
File naming

File names are generated automatically based on your product name, program name, and a UTC timestamp. This keeps your files organized and easy to track across versions.

Customer_Service__Engage_Connect__20250218_1324Z.xlsx
Important: Don't rename files manually. The naming convention is designed for version control, and changing it can make it harder to identify the current version.
Product and program names

Your product and program names appear next to the Design and Report headers. Click them at any time to open the edit modal and update the names. When no names are set, a placeholder Product Name: Training Program is shown.

When you load a file, any names stored in the Excel are imported automatically. If the file has no names, the placeholder appears and you can set them by clicking.

Name changes are covered by undo, so you can revert them if needed.

Loading a file

Go to File → Load and select a previously saved .xlsx file. The app reads the data and rebuilds your day tables, charts, and reports automatically.

Loading a file replaces everything currently in the app. Make sure to save your current work first if you haven't already.
Managing Your Plan

Filtering

Narrow down the activities shown in your day tables to focus on specific criteria. Filters affect only the day tables — the Report section always reflects the full, unfiltered data set.

Opening the filter

Click the Filter button in the top menu bar. It appears once you have at least one activity in your plan. A modal opens with all available filter options.

When filters are active, the Filter button shows a small blue dot indicator so you always know filtering is on.

Filter types

The filter modal groups filters by the type of data in each column.

Text searchType to search within Unit, Module, or Learning Objective. Matches are case-insensitive and partial — typing “sales” will match “Salesforce” or “Sales Coaching”.
Chip selectorsClick one or more chips to filter by Cognitive Task, Learner Activity, Delivery Method, Media, or Type of Content. Selecting multiple chips in a category shows rows matching any of the selected values.
Grouped chipsLearner Activity and Media chips are organized into named groups. Click a group chip to select or deselect all items in that group at once. When every item in a group is individually selected, the group chip highlights automatically.
PlanFilter by plan status (Keep, Remove, etc.). This section only appears when the Planner is active.
Duration rangeSet a minimum and/or maximum duration in minutes to show only activities within that range.
Filter bar

When filters are applied, a bar appears below the navigation showing your active filters as labeled tags. This gives you a quick summary of what’s currently filtered without reopening the modal.

Click Clear All on the right side of the filter bar to remove all filters at once. You can also clear all filters from within the modal using the Clear All Filters link at the bottom left.

Filtered duration

While filters are active, each day table’s Total Duration updates to reflect only the visible (matching) rows. The number turns blue and shows a (filtered) label so you can tell at a glance that you’re looking at a subset.

How breaks are handled

Break rows are only shown when your filters are purely duration-based. If any column-specific filter is active (such as Cognitive Task or Delivery Method), breaks are automatically hidden since those fields don’t apply to them.

Filters persist while you edit, drag, or add activities. They are automatically cleared when you reset the app or remove all activities.
Analysis

Report

A detailed breakdown of your training program that helps you spot patterns, imbalances, and opportunities to improve your design.

Getting there

Click the Report ↓ button in the top menu to jump straight to the Report section. Once you're there, the button changes to Design ↑ to bring you back to the day tables.

Numerical tables

Each report category (Cognitive Task, Learner Activity, Delivery Method, Media, Content Type) gets its own table showing:

CountHow many activities use that category.
DurationTotal minutes spent on that category.
Time %What percentage of total program time it represents.

The Learner Activity and Media tables include a Group column as the first column, showing which group each item belongs to. Column headers use the specific term for each table — Activity, Media, Method, Type, or Task — rather than a generic label.

Click Show Days above any table to filter by specific training days for a more focused view.

Donut charts

A quick visual snapshot of how time is distributed across each category. The center shows the dominant category and its percentage. Hover over any slice for details. Useful for spotting when one category dominates or another is underrepresented.

The Learner Activity and Media charts show group-level distribution — each of the six slices represents a group, not an individual item. The other charts (Cognitive Task, Delivery Method, Content Type) show individual item slices as before.

Line charts

These appear automatically when your program spans multiple days. The X-axis shows training days and the Y-axis shows minutes per category per day.

Use these to see how emphasis shifts over time. For example, you might check whether cognitive complexity increases as the program progresses, or whether instructor-led time tapers off as learners gain independence.

Content Type vs. Cognitive Tasks

A cross-reference table at the bottom of the report. It shows how your content types (Information, Procedural, Problem-Solving) map against each cognitive task level (Remember through Create). The Procedural and Problem-Solving columns distinguish between Software / Tools and Communication Skills subtypes. This helps you confirm that your program challenges learners appropriately across different types of content.

Excel Workflow

Excel Template

If you prefer to plan in Excel first, download the blank template. It comes with dropdown menus already built into the key columns, so your data stays compatible with the app.

Columns with dropdown validation

These columns have built-in list validation to keep your entries consistent when you load the file into Training Designer:

Cognitive Task Learner Activity Delivery Method Media Content Type Plan
Loading your Excel file into the app
  • Fill in the template in Excel. Each row represents one learning activity.
  • Save the file. Keep the column headers exactly as they are.
  • In Training Designer, go to File → Load and select your file.
  • The app reads your data and generates day tables, reports, and charts automatically.
Important: Do not rename, reorder, or delete any column headers. The load function maps your data by matching header names exactly.
